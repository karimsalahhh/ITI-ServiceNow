## How to run

1. Open the folder **E-Commerce-Final-Project** in VS Code (this folder contains `index.html`).
2. Use **Live Server** on `index.html`.

> Important: this project uses paths like `/assets/...` and `/partials/...`, so Live Server must be started from the project root.

## Tech used

- HTML + Bootstrap 5
- Vanilla JavaScript (ES Modules)
- Fetch API (Supabase REST)
- LocalStorage (Favourites + Cart)
